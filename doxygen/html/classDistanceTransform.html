Set the size of the distance transform mask, see the OpenCV enum cv::DistanceTransformMasks.

Parameters
[in]sizeSize of the mask, see cv::DistanceTransformMasks

The default mask size is cv::DIST_MASK_5

Note
In case of the DIST_L1 or DIST_C distance type, the parameter should be set to 3 because a 3×3 mask gives the same result as 5×5 or any larger aperture.
