void TemplateMatching::normalizeMatch ( TemplateMatch &  m)

Normalize an input match by setting its actual 6D position and no image offset.

Parameters
mAn intpu TemplateMatch

Given an input TemplateMatch instance, this function estimates the actual location of the object as if it had no image offset. The r_quat and t_vec are set accordling, while img_offset is set to (0,0). If the match has no image offset (i.e., img_offset is (0,0)) r_quat and t_vec are not changed.
