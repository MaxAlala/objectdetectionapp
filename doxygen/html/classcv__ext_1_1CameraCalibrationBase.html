Set the number of levels of the gaussian image pyramid used to extract corners.

Parameters

Build a gaussian pyramid of the image and start to extract the corners from the higher level of a gaussian pyramid. For large images, it may be useful to use a 2 or 3 levels pyramid. Set to 1 by default
