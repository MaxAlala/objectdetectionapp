Detailed Description

template<typename _T>
class cv_ext::Box3< _T >

Template class for 3D boxes.

Inspired by the OpenCV cv::Rect_ class for 2D rectangles, this class represent 3D boxes with faces aligned with the the x, y and/or z axes. It is described by the following parameters:

  • Coordinates of the two extreme vertices
  • Coordinates the origin (i.e., the "smallest" extreme point) plus width, height, and depth
Note
Differently from the cv:::Rect_ object, all boundaries of the box are inclusive, i.e., a box contains both its extreme vertices

For convenience, some Box aliases ara available, e.g.: cv_ext::Box3i, cv_ext::Box3f, and cv_ext::Box3d.
