void CannyEdgeDetector::enableRGBmodality ( bool  enable)
inline

Enable/disable the "RGB modality" of the edge detector.

Parameters
[in]enableIf true, apply Canny to each image channel

If enable is true, in case of an input RGB image, the Canny edge detector is applied to each image channel (RGB) and then each resulting edge map is merged (OR operator) into an unique edge map

Note
In case of grey level input images, the conventional Canny edge detector is applied even if the "RGB modality" is enabled.
